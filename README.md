## Setup
1. Clone this repository
   ```
   git clone https://github.com/martagorska-dev/classy-weather
   ```
2. Install dependencies
   ```
   npm install
   ```
3. Start application
   ```
   npm start
   ```


## Usage
- Enter the first three letters of the city in a search bar 🖊️
